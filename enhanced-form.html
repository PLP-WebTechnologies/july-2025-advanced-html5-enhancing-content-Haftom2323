Definition List: Web Terms

Semantic HTML
Using HTML elements that clearly describe their meaning in a human- and machine-readable way.
Responsive Design
An approach to web design that makes web pages render well on a variety of devices and window or screen sizes.
Progressive Enhancement
A strategy for web design that emphasizes core webpage content first, then adds more nuanced and technically rigorous layers of presentation and features.
